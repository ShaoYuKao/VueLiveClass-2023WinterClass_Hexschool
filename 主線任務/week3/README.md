## 主線任務

第三週同學可多練習 Vue.js 的各項語法及指令，目標以串接 API 完成一頁式的產品新增、刪除、修改的頁面。

- 可以新增、編輯、刪除商品
- 商品啟用、關閉可以使用不同的顏色標示

## 作業等級表

1. LV1：參考程式碼範例（需補上連結），並重新撰寫及補上註解（禁止複製範例程式碼）

1. LV3：完成 LV2 並且建立超過 10 個正式產品內容（不使用假資料）
